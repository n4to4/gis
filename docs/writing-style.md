# 文体ルール

## 言語

日本語で書く。専門用語は日本語・英語を併記する（例：ベクター（Vector））。

## 文体

- 体言止め・断定形を基本とする（「〜です」「〜ます」は使わない）
- 冗長な説明を避け、簡潔に
- 箇条書きを積極的に使う

**OK例：**
> ポリゴンで地物を表現する形式。面積や境界の計算に向いている。

**NG例：**
> ポリゴンで地物を表現する形式です。面積や境界の計算に向いています。

## 読者像

GISの知識はないが、プログラミングの基礎知識はある開発者・エンジニアを想定する。

- 測量・地理学の専門用語は初出時に簡単な説明を添える
- ツール・ライブラリ名（QGIS、MapLibre など）はそのまま使ってよい

## 用語の表記統一
